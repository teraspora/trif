Teraspora Fractals

About me, this website and these images...

Hi. I'm John Lynch. I love coding, playing with mathematics, and using both of these to create digital art. I also love playing music (piano, sax, tin whistles), growing vegetables, trees and even a few flowers; cooking and making chutneys, marmalades etc.; and enjoying good coffee, preferably freshly-ground and ideally not long roasted!

Back in the day, I did a mathematics degree at Hull University, since which I've worked with computers for many years in a variety of roles, mostly involving operations and technical support, as well as running a PC factory, but over the past few years my early passion for coding has, delightfully, returned. And I've been using it, and my mathematical knowledge, to create digital art, mainly using Java and Clojure, but lately also in GLSL, directly programming the GPU.

All this stuff is quite specialised, though, so in January 2018 I enrolled on a Code Institute course to learn all the technologies required to develop both the front- and back-ends of websites and web applications, and to understand all the necessary wiring that serves to knit all these technologies together.

This website is the culmination of over a year of study online. It is my fifth and final "Milestone Project" for the Code Institute course. It is built in Python 3.7 on the backend, with Django 2.2. The frontend templates are built using the Django template language, with Bootstrap 4 and my own CSS. It is hosted at Heroku. It uses a PostgreSQL database also hosted at Heroku to store image data and user profile data. The images themselves are stored in an Amazon S3 bucket, along with other static files.

Previous Milestone Projects for the course include

  • My own website, now hosted at www.teraspora.net
  • Teraspora Fractal Explorer, a Javascript app which allows the user to generate fractal images in the browser
  • Arith, a mental arithmetic game developed using Flask
  • West Cork Vegetables, a database-driven app using Flask and MongoDB, with full CRUD functionality, targeted at organic vegetable growers

So whence the images? Well, from an early age I have loved patterns.

The images themselves are the result of several years of developing and refining a Java program which generates "Escape-time Fractals" by iterating functions over the Complex plane. I found that by creating Complex functions which apply real trigonometric functions separately to the real and imaginary parts, and concocting more and more elaborate variations, composing functions, adding pre-transformations, varying a whole range of parameters and varying the algorithms I use to map iteration counts to colours, I could create some very interesting images indeed!

For more information see this short article on my website www.teraspora.net
